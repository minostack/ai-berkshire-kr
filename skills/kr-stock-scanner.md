# 한국 주식 종목 발굴 스캐너

$ARGUMENTS 조건으로 KOSPI / KOSDAQ 시장에서 투자 후보 종목 리스트를 생성합니다.

**지원 입력 형식**:

| 입력 | 예시 | 설명 |
|------|------|------|
| 테마/섹터 | `AI 반도체` `K-방산` `바이오` | 해당 테마 관련 상장사 스캔 |
| 시장 전체 | `KOSPI 대형주` `KOSDAQ 성장주` | 시장 전체에서 조건 필터링 |
| 조건 지정 | `ROE 15% 이상 저PBR` | 사용자 정의 재무 조건 |
| 비어 있음 | (입력 없음) | 기본 4대가 기준으로 전 섹터 스캔 |

---

## ⚠️ 데이터 수집 필수 규칙 (실행 전 반드시 확인)

Claude Code의 내장 Fetch 도구는 아래 사이트에서 **자동 차단**됩니다.
이 스킬은 반드시 아래 3가지 방법만 사용합니다:

| 방법 | 도구 | 용도 | 우선순위 |
|------|------|------|---------|
| **방법 1** | `python3 tools/kr_scanner.py` + `kstock_data.py` | DART API 재무 데이터 | 1순위 ✅ |
| **방법 2** | `WebSearch` 도구 | 종목 탐색, 정성 정보 수집 | 2순위 ✅ |
| **방법 3** | DART URL 안내 후 수동 확인 | 방법 1·2 실패 시 | 3순위 ✅ |
| **❌ 금지** | `Fetch(finance.naver.com/...)` | 직접 접속 — 차단됨 | 사용 금지 |
| **❌ 금지** | `Fetch(comp.fnguide.com/...)` | 직접 접속 — 차단됨 | 사용 금지 |

---

## 설계 철학

> "좋은 투자는 좋은 종목 발굴에서 시작된다. 그러나 대부분의 스크리너는 숫자만 보고 사업을 보지 않는다." — 버핏

이 스킬은 **2단계 필터** 방식입니다.

```
1단계: 정량 필터 → 재무 수치로 후보군 압축 (광범위하게)
2단계: 정성 필터 → 비즈니스 모델·경쟁우위 기준으로 최종 선별 (엄격하게)
```

숫자만 좋고 사업이 나쁜 종목은 제외합니다.
사업은 좋지만 아직 숫자가 부족한 성장주는 별도 트랙으로 분류합니다.

---

## 실행 흐름

### 0단계: 입력 분석 및 스캔 범위 확정

$ARGUMENTS를 분석해서 스캔 범위를 결정합니다:

- **테마/섹터 입력**: 해당 테마 KOSPI+KOSDAQ 상장사 WebSearch로 수집
- **시장 전체**: KRX 업종 분류 기준 전 섹터 커버 (시가총액 상위 300개 우선)
- **조건 지정**: 사용자 조건을 정량 필터에 반영
- **입력 없음**: 기본 조건으로 전 섹터 스캔

스캔 전 사용자에게 아래를 안내합니다:

```
📋 스캔 범위: {범위}
⏱️  예상 소요: 약 {N}분
🎯 목표 출력: 투자 후보 {N}~{M}개 종목 리스트
```

---

### 1단계: 후보군 수집

**방법 1 — DART API 전 상장사 재무비율 일괄 조회 (가장 정확, 먼저 시도)**

```bash
# 전 상장사 ROE·부채비율·영업이익률 일괄 조회 → 필터 자동 적용
python3 tools/kr_scanner.py dart-ratio --year 2024

# 특정 섹터 종목 리스트
python3 tools/kr_scanner.py sector {섹터명}

# 단일 종목 재무 요약
python3 tools/kr_scanner.py info {종목코드}
```

`dart-ratio`는 DART API 키가 있으면 전 상장사 재무비율을 한 번에 받아서
ROE ≥ 10%, 부채비율 ≤ 150%, 영업이익률 ≥ 8% 기준으로 자동 필터링합니다.

**방법 2 — WebSearch로 후보군 탐색 (방법 1 보완 또는 대체)**

`finance.naver.com`, `comp.fnguide.com`은 Fetch로 직접 접속이 차단됩니다.
반드시 WebSearch 도구로 검색엔진을 경유해서 데이터를 수집합니다:

```
WebSearch: "KOSPI {섹터명} 상장사 시가총액 ROE 순위 2024"
WebSearch: "KOSDAQ {테마명} 우량주 재무제표 영업이익률"
WebSearch: "{테마} 대장주 저PBR 고ROE 스크리닝"
WebSearch: "dart.fss.or.kr {섹터} 사업보고서 재무비율"
```

**방법 3 — DART 직접 접속 안내 (방법 1·2로 부족할 때)**

특정 종목의 원본 재무제표가 필요하면 URL을 사용자에게 안내합니다:

```
DART 사업보고서: https://dart.fss.or.kr/dsab001/main.do
KRX 업종별 시세: https://data.krx.co.kr/contents/MDC/STAT/standard/MDCSTAT03901.cmd
```

**수집 우선순위 요약**:
```
방법 1 (DART API) → 방법 2 (WebSearch) → 방법 3 (URL 안내)
❌ Fetch(naver/fnguide) 절대 사용 금지
```

### 2단계: 1차 정량 필터 (광범위 적용)

아래 기준으로 후보군을 압축합니다. **하나라도 통과하면 2단계 진입** (AND가 아닌 OR 조건 — 이 단계에서 너무 엄격하게 걸러내지 않음):

#### 가치주 트랙 (성숙 기업)

| 지표 | 기준 | 확인 방법 (우선순위 순) |
|------|------|----------------------|
| ROE | 최근 3년 평균 ≥ 10% | 방법1: `kr_scanner.py dart-ratio` → 방법2: WebSearch "{종목명} ROE 3년 평균" |
| 영업이익률 | 최근 3년 평균 ≥ 8% | 방법1: `kstock_data.py financials` → 방법2: WebSearch "{종목명} 영업이익률 추이" |
| 부채비율 | ≤ 150% (금융주 제외) | 방법1: `kr_scanner.py dart-ratio` → 방법2: WebSearch "{종목명} 부채비율 재무제표" |
| PBR | ≤ 1.5배 (저평가 기준) | 방법1: `kstock_data.py valuation` → 방법2: WebSearch "{종목명} PBR 현재" |
| 배당수익률 | ≥ 2% (선택 조건) | 방법1: `kstock_data.py quote` → 방법2: WebSearch "{종목명} 배당수익률" |
| 시가총액 | ≥ 500억원 (유동성 확보) | 방법1: `kstock_data.py quote` → 방법2: WebSearch "{종목명} 시가총액" |

#### 성장주 트랙 (고성장 기업)

| 지표 | 기준 | 확인 방법 (우선순위 순) |
|------|------|----------------------|
| 매출 성장률 | 최근 2년 연평균 ≥ 20% | 방법1: `kstock_data.py financials` → 방법2: WebSearch "{종목명} 매출 성장률 2년" |
| 매출총이익률 | ≥ 30% (사업모델 검증) | 방법1: `kstock_data.py financials` → 방법2: WebSearch "{종목명} 매출총이익률" |
| 영업현금흐름 | 최근 1년 (+) 전환 | 방법1: `kstock_data.py financials` → 방법2: WebSearch "{종목명} 영업현금흐름" |
| 시가총액 | ≥ 300억원 | 방법1: `kstock_data.py quote` → 방법2: WebSearch "{종목명} 시가총액" |

#### 즉시 제외 조건 (어느 트랙이든 해당되면 탈락)

```
❌ 관리종목 / 투자주의환기종목
❌ 최근 3년 내 감사의견 비적정
❌ 최대주주 지분 < 10% (경영진 이해관계 부재)
❌ 최근 2년 연속 영업손실 (성장주 트랙 제외 — 단, 매출 성장 조건 강화)
❌ 부채비율 > 400%
❌ 소송/횡령/분식회계 이슈 현재 진행 중
```

### 3단계: 2차 정성 필터 (엄격 적용)

1차 필터 통과 종목에 대해 아래 4가지 질문으로 정성 평가합니다.
**4개 중 3개 이상 YES여야 최종 후보 진입**:

```
Q1. 이 사업을 10년 후에도 이해할 수 있는가? (비즈니스 지속성)
Q2. 가격 결정권(Pricing Power)이 있는가? (경쟁 우위)
Q3. 경영진이 주주 친화적인가? (자본 배분 능력)
Q4. 현재 가격이 명백히 싸거나 적정한가? (안전마진)
```

각 질문은 **WebSearch 도구**로 수집한 정보로 판단합니다
(Fetch 직접 접속 금지 — WebSearch만 사용):

```
Q1 WebSearch: "{종목명} 산업 10년 전망 기술 변화 위협"
Q2 WebSearch: "{종목명} 영업이익률 추이 가격 인상 경쟁우위"
Q3 WebSearch: "{종목명} 자사주 매입 배당 증가 주주환원"
Q4 WebSearch: "{종목명} PER PBR 역사적 평균 저평가 밸류에이션"
```

### 4단계: 섹터별 경쟁 포지션 확인

최종 후보 종목은 같은 섹터 내 경쟁사와 비교합니다:

| 종목 | 시총 | PER | PBR | ROE | 영업이익률 | 매출성장 | 경쟁우위 |
|------|------|-----|-----|-----|-----------|---------|---------|
| A사  | Xbn | Xx | Xx | X% | X%      | X%     | 브랜드/기술/원가 |

섹터 내 상위 1~2위 포지션 종목을 우선합니다.

## 주요 주의사항

1. **이 리스트는 투자 권유가 아닙니다** — 심층 분석의 출발점입니다
2. **데이터 시점**: 스캔 일자 기준이며 실시간 변동을 반영하지 않습니다
3. **정보 충분도**: 소형주·신규상장주는 데이터 한계로 정성 평가 신뢰도가 낮을 수 있습니다
4. **편향 주의**: AI 스크리너는 미디어 노출이 많은 종목에 편향될 수 있습니다 — 비인기 우량주 가능성을 항상 열어두세요
5. **교차 검증 필수**: 핵심 재무 수치는 반드시 DART 원본 재무제표로 확인하세요
